# hvr

install
```
go get github.com/secgo/hvr/
```

للبحث عن نتائج موقع  هكر ون 
```bash
go run main.go -k phpinfo
```
OR
```bash
./hvr -k phpinfo
```

لعرض الخيارات 
```bash
go run main.go -h
# or
./hvr -h
```
نتائج هذا الامر 
```bash
go run main.go -k phpinfo
```
